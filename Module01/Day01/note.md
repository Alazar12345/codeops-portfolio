# Day 1 Notes

## Five Terminal Commands

1. pwd - Shows the current directory.
2. ls - Lists the files and folders.
3. cd - Changes to another directory.
4. mkdir - Creates a new folder.
5. rm - Removes a file.

## Three Git Commands

1. git init - Creates a new Git repository.
2. git status - Shows the current status of the repository.
3. git commit - Saves changes with a message.
